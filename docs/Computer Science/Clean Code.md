# <b>Clean Code</b>

---
클린코드는 읽기 쉽고 유지하기 쉬운 코드를 말합니다.<br>
클린코드는 좀 더 확장 가능하고 아래의 정해둔 기준에 맞춰야 합니다.
- 네이밍 컨벤션
- 가독성을 위한 포맷(일관된 들여쓰기, 적정한 공백 및 주석)
- 작고 읽기 쉬운 클래스와 메서드
- 부연 설명이 필요없는 주석
