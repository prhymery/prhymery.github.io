## .NET

.NET은 **다양한 유형의 애플리케이션을 구축하기 위해 Microsoft에서 만든 오픈 소스 개발자 플랫폼**입니다.
즉, 마이크로소프트에서 개발한 **Windows 프로그램 개발 및 실행 환경(프레임워크)**입니다.

### .NET 특징

- C#, F# 또는 Visual Basic으로 .NET 앱을 작성할 수 있습니다.
- 크로스 플랫폼이 가능하다.
- 공통적인 기본 .NET 클래스 라이브러리[^NET-Class-Library] 및 API를 제공합니다.
- .NET 패키지 에코시스템인 NuGet[^NuGet]을 유지 관리합니다.

[^NET-Class-Library]: **.NET 클래스 라이브러리** : 다양한 응용 프로그램에서 사용할 수 있는 모듈을 제공합니다.
[^NuGet]: **NuGet** : .NET 용 무료 오픈 소스 패키지 관리자 프로그램이다. <br> 다른 개발자가 프로젝트에서 사용할 수 있도록 만든 재사용 가능한 코드가 포함되어 있습니다.

## .NET Framework

**Windows에서 애플리케이션을 빌드하고 실행하기 위한 소프트웨어 개발 프레임워크**입니다. <br>
다양한 유형의 애플리케이션을 구축하기 위한 라이브러리로 구성된 개발자 플랫폼으로 .NET 코드를 Linux, macOS, Windows, iOS, Android 등 **다양한 환경에서 실행할 수 있습니다**.

### .NET Framework 특징

두가지 주요 구성 요소가 있습니다.

1. **CLR**(**C**ommon **L**anguage **R**untime, 공용 언어 런타임)
   - 실행 중인 앱을 처리하는 실행 엔진으로 스레드 관리, 가비지 수집, 형식 안정성, 예외 처리드으이 서비스를 제공합니다.
2. **FCL**(**F**ramework **C**lass **L**ibrary, 프레임워크 클래스 라이브러리)
   - 공통 기능에 대한 타입(strings, datas, numbers 등)과 API(파일 읽고 쓰기, 데이터베이스 연결, 그리기 등)를 제공합니다.

### .NET Framework 아키텍처

1. C#, F#, Visual Basic 으로 .NET Framework 응용 프로그램을 생성합니다.
2. 응용 프로그램에 작성된 코드는 CIL(**C**ommon **I**ntermediate **L**anguage)로 컴파일되고 .dll, .exe 확장자 파일로 저장됩니다.
3. 앱이 실행될 때 CLR이 어셈블리를 가져와 JIT(**J**ust **I**n **T**ime) 컴파일러를 사용하여 CIL을 앱이 실행 중인 컴퓨터의 특정 아키텍쳐에서 실행할 수 있는 기계어로 변환합니다.
